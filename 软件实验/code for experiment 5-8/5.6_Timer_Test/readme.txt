
===========================================================================================
Timer_Test实验
===========================================================================================
本实验实现看门狗定时器的计时和其中断。本例程有两种运行方式。
Timer_Test IN RAM         将程序下载到SDRAM中调试。
Timer_Test IN Flash       将程序烧写到Nor Flash中。上电自动运行。
